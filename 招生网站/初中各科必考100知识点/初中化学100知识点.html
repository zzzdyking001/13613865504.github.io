一、基本概念与理论 约15个核心点

化学学科的基础语言和核心概念。

1. 物质的变化与性质

1. 物理变化（无新物质生成，如：粉碎、蒸发、溶解）
2. 化学变化（有新物质生成，如：燃烧、锈蚀、食物腐烂）
3. 物理性质（颜色、状态、气味、熔点、沸点、密度、硬度等）
4. 化学性质（可燃性、稳定性、酸性、碱性、氧化性等）

2. 物质的构成与组成

5. 分子（保持物质化学性质的最小微粒）
6. 原子（化学变化中的最小微粒）——原子由原子核（质子+中子）和核外电子构成
7. 离子（带电的原子或原子团）
8. 元素（质子数相同的同一类原子的总称）
9. 元素符号与原子序数
10. 相对原子质量（Ar = 一个原子的实际质量 / (1/12 × 一个C-12原子的质量)）

3. 化学用语

11. 化学式（如：H₂O、NaCl）
12. 化合价（常见元素化合价口诀）
13. 化学方程式（书写原则：客观事实 + 质量守恒）
14. 质量守恒定律：参加反应的各物质质量总和等于反应后生成的各物质质量总和
15. 根据化学方程式的计算（步骤：设、写、列、求、答）

二、身边的化学物质 约25个核心点

常见物质的性质、用途和实验。

1. 空气与氧气

16. 空气组成：N₂（78%）、O₂（21%）、稀有气体（0.94%）、CO₂（0.03%）
17. 氧气的物理性质：无色无味、不易溶于水、密度比空气略大
18. 氧气的化学性质：助燃性、氧化性（支持燃烧）
19. 氧气的制备（实验室）：加热KMnO₄、H₂O₂ + MnO₂、加热KClO₃ + MnO₂
20. 氧气的验满方法：带火星的木条放在集气瓶口
21. 空气中氧气含量测定（红磷燃烧实验）

2. 水

22. 水的组成（电解水实验：正极O₂、负极H₂，体积比1:2）
23. 水的净化方法：沉降、过滤、吸附、蒸馏
24. 硬水与软水的区分（肥皂水——浮渣多的是硬水）
25. 硬水软化方法：煮沸、蒸馏

3. 碳及其化合物

26. 碳的单质：金刚石（正八面体）、石墨（层状）、C₆₀（足球状）
27. 碳的化学性质：稳定性、可燃性、还原性（C + 2CuO → 2Cu + CO₂↑）
28. CO₂的物理性质：无色无味、密度比空气大、能溶于水
29. CO₂的化学性质：不助燃、不可燃、与碱反应
30. CO₂的制备（实验室）：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
31. CO的性质：有毒、可燃性、还原性
32. CO₂与CO的鉴别方法

4. 金属与金属材料

33. 金属的物理共性（导电性、导热性、延展性、金属光泽）
34. 金属活动性顺序：K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au
35. 金属与酸反应：活泼金属 + 酸 → 盐 + H₂↑
36. 金属与盐溶液反应：前置后，盐可溶
37. 铁的冶炼原理：Fe₂O₃ + 3CO → 2Fe + 3CO₂
38. 铁的锈蚀条件：与氧气和水同时接触
39. 防止金属腐蚀的方法（保持干燥、涂油、刷漆、镀金属）

5. 酸、碱、盐

40. 酸的化学通性（5条）：使石蕊变红、与活泼金属反应、与金属氧化物反应、与碱反应、与盐反应
41. 碱的化学通性（4条）：使石蕊变蓝、使酚酞变红、与非金属氧化物反应、与酸反应、与盐反应
42. 盐的定义：由金属离子（或NH₄⁺）和酸根离子组成的化合物
43. 复分解反应的条件：有沉淀、气体或水生成
44. 常见酸碱盐的俗名与化学式（烧碱NaOH、纯碱Na₂CO₃、小苏打NaHCO₃、石灰石CaCO₃）
45. pH值与酸碱性的关系：pH=7中性、pH<7酸性、pH>7碱性

三、化学实验 约20个核心点

实验操作、常见仪器和气体检验。

1. 基本操作与仪器

46. 酒精灯的使用：禁止用嘴吹灭，禁止向燃着的酒精灯内添加酒精
47. 药品取用规则（粉末用纸槽、块状用镊子）
48. 量筒读数：视线与凹液面最低处保持水平
49. 天平的使用（左物右码）
50. 过滤操作“一贴二低三靠”
51. 蒸发操作（蒸发皿、玻璃棒搅拌、停止加热）
52. 常见仪器的名称和用途（试管、烧杯、量筒、集气瓶、酒精灯等）

2. 气体检验与实验

53. O₂的检验方法：带火星的木条复燃
54. CO₂的检验方法：通入澄清石灰水变浑浊
55. H₂的检验方法：点燃，淡蓝色火焰，烧杯内壁有水珠
56. 实验室制O₂、CO₂、H₂的装置选择（固+固加热/固+液不加热）
57. 排水法收集气体的适用条件（不溶于水、不与水反应）
58. 向上排空气法的适用条件（密度比空气大）
59. 装置气密性检查方法

3. 实验探究与推断

60. 酸碱中和反应的实验探究（指示剂选择）
61. 常见离子的检验（Cl⁻、SO₄²⁻、CO₃²⁻）
62. 物质的鉴别（利用颜色、气味、溶解性、化学性质等）
63. 物质的推断（突破口：特殊颜色、特殊现象、特殊反应）
64. 实验方案的设计与评价

四、化学与社会 约10个核心点

化学与生活、能源、环境。

65. 燃烧的条件：可燃物、助燃物（O₂）、温度达到着火点
66. 灭火的原理：清除可燃物、隔绝O₂、降温至着火点以下
67. 化石燃料（煤、石油、天然气——不可再生能源）
68. 新能源：太阳能、风能、氢能、核能等
69. 三大化石燃料燃烧对环境的影响（温室效应、酸雨、粉尘）
70. 温室气体（CO₂、CH₄等）与全球变暖
71. 酸雨的形成（SO₂、NO₂溶于水）
72. 常见有机合成材料（塑料、合成橡胶、合成纤维）
73. 六大营养素（糖类、油脂、蛋白质、维生素、无机盐、水）
74. 化学与人体健康（微量元素、食品安全）

五、常考化学方程式 约16个核心点

必须熟练掌握的化学方程式。

1. 氧气相关

75. 2H₂O₂ →（MnO₂）→ 2H₂O + O₂↑
76. 2KMnO₄ →（加热）→ K₂MnO₄ + MnO₂ + O₂↑
77. 2KClO₃ →（MnO₂，加热）→ 2KCl + 3O₂↑
78. C + O₂ →（点燃）→ CO₂
79. 2Mg + O₂ →（点燃）→ 2MgO
80. 3Fe + 2O₂ →（点燃）→ Fe₃O₄

2. 碳及化合物相关

81. C + 2CuO →（高温）→ 2Cu + CO₂↑
82. CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
83. CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O
84. 2CO + O₂ →（点燃）→ 2CO₂
85. CO + CuO →（加热）→ Cu + CO₂

3. 金属相关

86. Fe + CuSO₄ → FeSO₄ + Cu（铁表面出现红色固体）
87. Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag（铜表面出现银白色固体）
88. Fe₂O₃ + 3CO →（高温）→ 2Fe + 3CO₂

4. 酸碱盐相关

89. NaOH + HCl → NaCl + H₂O（中和反应）
90. 2NaOH + CO₂ → Na₂CO₃ + H₂O
91. Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑

5. 水相关

92. 2H₂O →（通电）→ 2H₂↑ + O₂↑

六、常考实验现象与颜色 约8个核心点

推断题和实验题的突破口。

93. 常见沉淀颜色：Cu(OH)₂（蓝色）、Fe(OH)₃（红褐色）、BaSO₄/AgCl（白色沉淀，不溶于酸）
94. 常见溶液颜色：CuSO₄（蓝色）、FeCl₃（黄色）、FeSO₄（浅绿色）
95. 火焰颜色：H₂（淡蓝色）、CO（蓝色）、CH₄（蓝色）、S在O₂中（蓝紫色）
96. 指示剂变色：石蕊（酸红碱蓝）、酚酞（酸无碱红）
97. 常见气体颜色：NO₂（红棕色）、Cl₂（黄绿色）、O₂/CO₂/H₂/CO（无色）
98. 常见固体颜色：CuO（黑色）、MnO₂（黑色）、Fe₃O₄（黑色）、Cu（紫红色）、Fe₂O₃（红棕色）
💡 复习小贴士
  • 化学方程式必须过关：中考化学的核心是化学方程式，建议每天默写5~8个，做到“看到反应物就能写出生成物和配平”。
  • 实验题注意“前后呼应”：实验题的答案往往藏在题干中，注意分析每一步操作的目的和现象描述。
  • 推断题要抓“突破口”：特殊颜色、特殊现象（如冒气泡、生成沉淀）、特殊反应条件是推断题的关键线索。
  • 规范化学用语：化学式大小写要分清（CO是一氧化碳，Co是钴），方程式要配平、标注反应条件和↑↓符号。
