一、声学 约5个核心点

声音的产生、传播与特性。

1. 声音的产生：一切发声的物体都在振动（振动停止，发声停止）
2. 声音的传播：需要介质（固体、液体、气体），真空不能传声
3. 声速：v固 > v液 > v气，空气中声速约为340 m/s
4. 声音的三个特性：音调（频率决定）、响度（振幅决定）、音色（材料结构决定）
5. 超声波与次声波（人耳可听频率范围：20~20000 Hz）

二、光学 约15个核心点

光的传播、反射、折射与透镜成像。

1. 光的直线传播

6. 光在同种均匀介质中沿直线传播
7. 光速：v真空 = 3×10⁸ m/s（空气中近似等于真空）
8. 应用：小孔成像、影子的形成、日食/月食

2. 光的反射

9. 反射定律：三线共面、两线分居、两角相等（反射角 = 入射角）
10. 镜面反射与漫反射（都遵循反射定律）
11. 平面镜成像特点：等大、等距、正立、虚像、左右相反

3. 光的折射

12. 折射规律：光从空气斜射入水/玻璃中，折射角小于入射角（反之则大于）
13. 现象：筷子在水面“折断”、池水看起来“变浅”、海市蜃楼
14. 光的色散：牛顿——太阳光经三棱镜分解成红橙黄绿蓝靛紫

4. 透镜

15. 凸透镜（中间厚、边缘薄）——会聚光线
16. 凹透镜（中间薄、边缘厚）——发散光线
17. 凸透镜成像规律：u > 2f（倒立、缩小、实像）——照相机
18. f < u < 2f（倒立、放大、实像）——投影仪
19. u < f（正立、放大、虚像）——放大镜
20. 眼睛与眼镜：近视眼（凹透镜矫正），远视眼（凸透镜矫正）

三、热学 约15个核心点

温度、物态变化、内能与热机。

1. 温度与物态变化

21. 温度计原理：液体热胀冷缩
22. 摄氏温度：冰水混合物为0℃，标准大气压下沸水为100℃
23. 六种物态变化：熔化、凝固、汽化、液化、升华、凝华
24. 晶体熔化规律：达到熔点后继续吸热但温度保持不变（如冰熔化）
25. 蒸发与沸腾的异同（蒸发在任何温度下都能进行）
26. 液化方式：降低温度、压缩体积

2. 内能与热机

27. 内能：物体内部所有分子做无规则运动的动能和分子势能的总和
28. 改变内能的两种方式：做功（搓手取暖）和热传递（晒太阳）
29. 比热容：c水 = 4.2×10³ J/(kg·℃)（水的比热容大，用于冷却/取暖）
30. 热值：1kg某种燃料完全燃烧放出的热量（Q = mq）
31. 热机四冲程：吸气、压缩、做功、排气（做功冲程将内能转化为机械能）

四、力学 约35个核心点

力与运动、压强、浮力、功与机械，中考重中之重。

1. 机械运动

32. 参照物：判断物体运动与静止时选作标准的物体
33. 速度：v = s / t（单位：m/s、km/h）
34. 匀速直线运动：速度保持不变
35. 平均速度：v = s总 / t总

2. 力

36. 力的作用效果：改变物体运动状态、使物体发生形变
37. 力的三要素：大小、方向、作用点
38. 力的相互作用：物体间力的作用是相互的
39. 重力：G = mg（g取9.8 N/kg，方向竖直向下）
40. 弹力：弹簧测力计的原理（在弹性限度内，弹簧的伸长量与所受拉力成正比）
41. 摩擦力：增大压力、增大接触面粗糙程度→增大摩擦；减小压力、减小粗糙程度、变滑动为滚动→减小摩擦

3. 力与运动

42. 牛顿第一定律（惯性定律）：一切物体在没有受到力的作用时，总保持静止状态或匀速直线运动状态
43. 惯性：一切物体都具有惯性（只与质量有关）
44. 二力平衡条件：同体、等大、反向、共线
45. 合力与分力：同向相加，反向相减

4. 压强

46. 压强：p = F / S（单位：Pa）
47. 增大/减小压强的方法（增大压力/减小受力面积→增大压强）
48. 液体压强：p = ρgh（与深度和液体密度有关）
49. 连通器原理：同种液体静止时液面相平
50. 大气压强（马德堡半球实验证明其存在，托里拆利实验测出数值）
51. 流体压强与流速的关系：流速大的位置压强小（飞机的升力）

5. 浮力

52. 浮力产生原因：液体对物体上下表面的压力差
53. 阿基米德原理：F浮 = G排 = ρ液gV排
54. 物体的浮沉条件：F浮 > G（上浮）、F浮 = G（悬浮/漂浮）、F浮 < G（下沉）
55. 应用：轮船（空心增大浮力）、潜水艇（改变自身重力）、密度计

6. 功与机械能

56. 功：W = Fs（单位：J，有力且有沿力方向的距离才做功）
57. 功率：P = W / t = Fv（单位：W）
58. 动能：物体由于运动而具有的能（质量越大、速度越大，动能越大）
59. 重力势能：物体由于被举高而具有的能（质量越大、高度越高，重力势能越大）
60. 弹性势能：与弹性形变程度有关
61. 机械能守恒：动能和势能可以相互转化（忽略摩擦阻力时守恒）

7. 简单机械

62. 杠杆平衡条件：F1l1 = F2l2
63. 杠杆分类：省力杠杆（l1>l2）、费力杠杆（l12）、等臂杠杆
64. 滑轮组：F = G/n（n为承重绳段数）
65. 机械效率：η = W有用 / W总 × 100%

五、电磁学 约30个核心点

电学、电路、欧姆定律、电功率、磁学，中考压轴题的高频区域。

1. 电路基础

66. 电荷：正电荷（质子）、负电荷（电子）——同种电荷相互排斥，异种电荷相互吸引
67. 电流：正电荷定向移动的方向为电流方向（与电子移动方向相反）
68. 电路组成：电源、导线、开关、用电器
69. 电路状态：通路、断路、短路
70. 串联电路与并联电路（串流并压——串联电流相等，并联电压相等）

2. 电学基本物理量

71. 电流（I）：单位A（安培）——电流表串联接入
72. 电压（U）：单位V（伏特）——电压表并联接入
73. 电阻（R）：单位Ω（欧姆）——导体对电流的阻碍作用
74. 影响电阻的因素：材料、长度（越长电阻越大）、横截面积（越粗电阻越小）、温度
75. 变阻器（滑动变阻器——通过改变接入电路中的电阻丝长度来改变电阻）

3. 欧姆定律

76. 欧姆定律：I = U / R（适用于纯电阻电路）
77. 串联电路规律：I总 = I1 = I2，U总 = U1 + U2，R总 = R1 + R2
78. 并联电路规律：I总 = I1 + I2，U总 = U1 = U2，1/R总 = 1/R1 + 1/R2

4. 电功与电功率

79. 电功（W）：W = UIt（单位：J，生活中常用kW·h）
80. 电功率（P）：P = UI = I²R = U²/R（单位：W）
81. 额定功率与实际功率（灯泡的亮度由实际功率决定）
82. 焦耳定律：Q = I²Rt（电流的热效应）

5. 家庭电路与安全用电

83. 家庭电路组成：火线、零线、地线
84. 保险丝：电阻大、熔点低（串联在火线上）
85. 安全用电原则：不接触低压带电体，不靠近高压带电体
86. 触电急救：切断电源或用绝缘体将电线挑开

6. 磁学

87. 磁体：指南针指向南北（地磁场）
88. 磁感线：从N极出发，回到S极（疏密表示磁场强弱）
89. 电流的磁效应：奥斯特实验（通电导线周围存在磁场）
90. 电磁铁：通电螺线管+铁芯（磁性强弱与电流大小、线圈匝数有关）
91. 电动机：通电导体在磁场中受力的作用（电能→机械能）
92. 发电机：电磁感应现象（机械能→电能，法拉第）
