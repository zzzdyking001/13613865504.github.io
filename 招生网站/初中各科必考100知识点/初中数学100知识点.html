一、数与代数 约35个核心点

贯穿初中数学始终，是后续学习的基础。

1. 数与式

▸ 实数

1. 有理数（整数、分数）与无理数（π、√2）
2. 数轴（三要素：原点、正方向、单位长度）
3. 相反数、倒数、绝对值
4. 科学记数法（a × 10ⁿ，1 ≤ |a| < 10）
5. 平方根（±√a）、算术平方根（√a）、立方根（∛a）
6. 实数的大小比较

▸ 整式

7. 整式加减（合并同类项）
8. 幂的运算：aᵐ·aⁿ = aᵐ⁺ⁿ，(aᵐ)ⁿ = aᵐⁿ，(ab)ⁿ = aⁿbⁿ
9. 整式乘法：单项式×多项式、多项式×多项式
10. 乘法公式：(a+b)² = a²+2ab+b² · (a-b)² = a²-2ab+b² · (a+b)(a-b) = a²-b²
11. 因式分解：提公因式法、公式法、十字相乘法
📌 核心公式：(a+b)² = a² + 2ab + b²　·　(a-b)² = a² - 2ab + b²　·　a² - b² = (a+b)(a-b)

▸ 分式

12. 分式有意义（分母 ≠ 0）
13. 分式的基本性质（约分、通分）
14. 分式运算（加减乘除）
15. 分式方程（增根检验）

▸ 二次根式

16. 二次根式有意义（被开方数 ≥ 0）
17. √a · √b = √(ab)（a≥0, b≥0）
18. √a / √b = √(a/b)（a≥0, b>0）
19. 最简二次根式（化简）

2. 方程与不等式

▸ 方程

20. 一元一次方程（ax + b = 0）
21. 二元一次方程组（代入消元 / 加减消元）
22. 一元二次方程：ax² + bx + c = 0（a ≠ 0）
23. 一元二次方程求根公式：x = [-b ± √(b²-4ac)] / 2a
24. 判别式 Δ = b² - 4ac（Δ>0 两不等实根；Δ=0 两相等实根；Δ<0 无实根）
25. 韦达定理：x₁+x₂ = -b/a，x₁·x₂ = c/a
26. 分式方程（增根检验）
27. 列方程解应用题（行程、工程、利润、增长率等）

▸ 不等式

28. 不等式性质（乘除负数改变方向）
29. 一元一次不等式（组）
30. 不等式组解集的数轴表示

3. 函数

▸ 平面直角坐标系

31. 各象限内点的坐标特征（+，+ / -，+ / -，- / +，-）
32. 对称点坐标规律（关于x轴、y轴、原点对称）

▸ 一次函数

33. 一般式：y = kx + b（k≠0）
34. k 决定增减性（k>0 增，k<0 减）
35. b 决定与y轴交点（0, b）
36. 待定系数法求解析式
37. 正比例函数 y = kx（过原点）

▸ 反比例函数

38. 一般式：y = k/x（k≠0）
39. 图像性质：k>0 一三象限，k<0 二四象限
40. 每个象限内 y 随 x 的增大而减小/增大

▸ 二次函数

41. 一般式：y = ax² + bx + c（a≠0）
42. 顶点式：y = a(x-h)² + k（顶点 (h,k)）
43. 对称轴：x = -b/(2a)
44. a 决定开口方向（a>0 向上，a<0 向下）和大小
45. 顶点坐标：(-b/(2a), (4ac-b²)/(4a))
46. 与x轴交点个数（判别式 Δ 决定）
47. 二次函数图像平移规律（上加下减，左加右减）

二、图形与几何 约30个核心点

培养空间观念和逻辑推理能力。

1. 基本图形

▸ 线与角

48. 线段、射线、直线（公理：两点确定一条直线）
49. 线段中点、角平分线
50. 余角、补角、对顶角（对顶角相等）
51. 平行线的判定与性质（同位角/内错角/同旁内角）
52. 垂线（垂线段最短）

▸ 三角形

53. 三角形内角和定理（180°）
54. 三角形外角定理（等于不相邻两内角和）
55. 三边关系（两边之和大于第三边）
56. 全等三角形判定：SSS · SAS · ASA · AAS · HL（Rt△）
57. 等腰三角形性质（等边对等角、三线合一）
58. 等边三角形性质（三边相等、三角=60°）
59. 直角三角形性质（勾股定理：a²+b²=c²）
60. 直角三角形判定（勾股定理逆定理）
61. 三角形中位线定理（平行于第三边且等于第三边的一半）

▸ 四边形

62. 平行四边形性质与判定（对边平行且相等、对角相等、对角线互相平分）
63. 矩形（对角线相等，四个直角）
64. 菱形（四边相等，对角线互相垂直平分）
65. 正方形（四边相等、四角为直角、对角线相等且垂直平分）
66. 梯形（一组对边平行）

▸ 圆

67. 圆心角、圆周角、弧、弦、弦心距
68. 垂径定理（垂直于弦的直径平分弦及弧）
69. 圆周角定理（同弧所对圆周角等于圆心角一半）
70. 切线的判定与性质（过圆心垂直于切线）
71. 圆的内接四边形（对角互补）

2. 图形变换

72. 轴对称（垂直平分线）
73. 中心对称（旋转180°）
74. 平移（方向和距离）
75. 旋转（旋转中心、旋转角）
76. 相似三角形判定：AA · SAS · SSS
77. 相似三角形性质（对应边成比例、面积比=相似比²）

3. 解直角三角形

78. 锐角三角函数：sin A = 对边/斜边，cos A = 邻边/斜边，tan A = 对边/邻边
79. 特殊角三角函数值（30°、45°、60°）
80. 解直角三角形应用（仰角、俯角、坡度、方位角）

💡 复习小贴士
  • 回归基础概念与公式：公式不要死记硬背，要理解其推导过程和适用条件。
  • 刷真题与错题本：近3~5年本地中考真题最有参考价值，错题本要记下错误原因和正确思路。
  • 几何题多总结模型：常见的辅助线添法和几何模型（如“手拉手”“一线三等角”）值得专门整理。
  • 函数题注意数形结合：画图可以帮助理解函数性质，也是解决复杂题的重要方法。
